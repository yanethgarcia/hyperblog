Examen Git y Github

Alumn@: [ronald santos]

  1. Que es un sistema de control de versiones?

    [Son una herramienta, con la finalidad de mantener un rastreo del estado de los cambios, a lo largo del desarrollo del proyecto]

  2. Cual es la diferencia entre git y github?

    [Git is an open-source tool, capable of monitoring changes in source code, while GitHub is a sore of community where developers can share developments]

  3. Cómo creo una rama en git?

    [With the use of the command git checkout -b [name of the branch]

  4. Cómo clono un repositorio?

    [Using the command clone follow by the https or ssh (URL)]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [Write git log and press enter, on Git Bash]

  6. Cómo puedo gener un tag?

    [git tag (name of the tag)]

  7. Cómo hago una mezcla de ramas?

    [using the command git merge (name of the branch from where the changes are coming from)]

  8. Cómo puedo moverme a un commit especifico?

    [Using command git log and while knowing the hash of the commit we want to get back to, just use git commit (hash of the commit you need)]

  9. Cómo puedo borrar una rama?

    [git branch -d (branch)]

  10. Cómo puedo borrar un tag?

    [git tag -d (tag we want to delete)]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]

  12. Para que sirve el comando git remote -v?

    [It shows the URL been use by the repository]

  13. Cómo le digo a git cual es mi email y correo?

    [git config --global user.email johndoe@example.com]

  14. Cuales son los pasos para conectar git con github?

    [pull request in github, new pull request, compare master with branch where the changes were made, create pull request]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.(letters of the alias) (command to be use by the alias)]
