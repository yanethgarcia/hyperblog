Examen Git y Github

Alumn@: [Carlos Esteban Maldonado Maldonado]

  1. Que es un sistema de control de versiones?

    [Es una herramienta que ayuda a los equipos de software a llevar un control en los cambios de su codigo]

  2. Cual es la diferencia entre git y github?

    [Git es un sistema de control de versiones y GitHub es una plataforma que admite repositorios git]

  3. Cómo creo una rama en git?

    [git checkout -b {nombre_rama}]

  4. Cómo clono un repositorio?

    [git clone http://github/repo/repo.git]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull origin {rama}]

  6. Cómo puedo gener un tag?

    [git tag -a {tag}]

  7. Cómo hago una mezcla de ramas?

    [git merge {rama_que_quiero_unir}]

  8. Cómo puedo moverme a un commit especifico?

    [git checkout {SHA}]

  9. Cómo puedo borrar una rama?

    [git branch -d {nombre_rama}]

  10. Cómo puedo borrar un tag?

    [git tag -d {nombre_tag}]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]

  12. Para que sirve el comando git remote -v?

    [Listar las conexiones remotas con los repositorios]

  13. Cómo le digo a git cual es mi email y usuario?

    [git config --global user.email hola@mail.com git config --global user.password pass]

  14. Cuales son los pasos para conectar git con github?

    [Se puede autenticar por token de acceso personal]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.{alias} {comando}]
