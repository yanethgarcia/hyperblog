Examen Git y Github

Alumn@: [Cecilia Beltrame]

  1. Que es un sistema de control de versiones?

    [Un sistema de control de versiones es una herramienta de software para hacer el seguimiento de los cambios en un codigo fuente en un equipo]

  2. Cual es la diferencia entre git y github?

    [Github es un repositorio y git es un sistema de control de versiones]

  3. Cómo creo una rama en git?

    [git branch {new_branch_name]

  4. Cómo clono un repositorio?

    [git clone git@ip_del_server:nombre:repo.git]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull]

  6. Cómo puedo gener un tag?

    [En consola, git tag -a v1.4 -m 'my version 1.4']

  7. Cómo hago una mezcla de ramas?

    [Usando 'merge, en consola: ejemplo git checkout master y luego git merge prueba]

  8. Cómo puedo moverme a un commit especifico?

    [Obteniendo el hash del commit a traves del git log]

  9. Cómo puedo borrar una rama?

    [Para eliminar una rama de nuestro repositorio local, git branch -d {branch_name} en consola]

  10. Cómo puedo borrar un tag?

    [Si la etiqueta no fue publicada, se utiliza 'git tag -d {nombre}' en consola]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git diff- en consola]

  12. Para que sirve el comando git remote -v?

    [Sirve para obtener la a que repositorio remoto esta conectado el repositorio local]

  13. Cómo le digo a git cual es mi email y correo?

    [En consola. git user config user.name 'usuario' para decirle cual es mi usuario. Para decirle cual es mi correo hay que escribir en consola 'git config user.email 'email']

  14. Cuales son los pasos para conectar git con github?

    [En primer lugar hay que crear el nuevo remoto con 'git remote add origin'. Luego agregar los cambios con git add, asi como tambien hay que hacer git commit a los cambios realizados. Luego git push al repositorio de github]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.co alias]
