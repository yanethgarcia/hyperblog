Examen Git y Github

Alumn@: [David Villegas]

  1. Que es un sistema de control de versiones?

    [Es un sistema que permite guardar diferentes actualizaciones del código fuente de un software y la que permite restaurar a una versión anterior en tal caso que presente algún inconveniente la versión actual]

  2. Cual es la diferencia entre git y github?

    [Git es una aplicación que se instala y se usa de forma local para el manejo de control de versiones, Github es una plataforma web igualmente de manejo de control de versiones en donde los repositorios van a un servidor en la nube y ahí se guardan los cambios que se generan en el código fuente y puede ser utilizado con la aplicación Git.]

  3. Cómo creo una rama en git?

    [git branch -b nombre_de_la_rama]

  4. Cómo clono un repositorio?

    [git clon repositorio (puede ser por ssh o por https)]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull origin nombre_de_la_rama]

  6. Cómo puedo gener un tag?

    [git tag -a nombre_del_tag]

  7. Cómo hago una mezcla de ramas?

    [git merge]

  8. Cómo puedo moverme a un commit especifico?

    [git revert]

  9. Cómo puedo borrar una rama?

    [git branch -d nombre_de_la_rama]

  10. Cómo puedo borrar un tag?

    [git tag -d nombre_del_tag]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]

  12. Para que sirve el comando git remote -v?

    [Muestra los repositorios remotos]

  13. Cómo le digo a git cual es mi email y correo?

    [git config user.name git config user.email (Para configurar nombre y correo)]

  14. Cuales son los pasos para conectar git con github?

    [Se debe crear una llave en git localmente para conectarse vía ssh, unavez creada la llave, se debe agregar en las configuraciones del usuario (SSH Keys) en github y listo]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.XX comando]
