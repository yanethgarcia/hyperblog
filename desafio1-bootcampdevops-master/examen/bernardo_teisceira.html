Examen Git y Github

Alumn@: Bernardo Teisceira

  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones, como indica su nombre es un programa que permite realizar versionado de código.

  2. Cual es la diferencia entre git y github?

    Git es el controlador de versiones y GitHub es una plataforma que permite alojar repositorios Git.

  3. Cómo creo una rama en git?

    git checkout *nombre rama nueva*

  4. Cómo clono un repositorio?

    git clone git@ip_del_server:nombre_repo.git

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    git pull origin *nombre_rama*

  6. Cómo puedo gener un tag?

    git tag *tag*

  7. Cómo hago una mezcla de ramas?

    git merge *nombre_rama*

  8. Cómo puedo moverme a un commit especifico?

    git commit *hash_de_commit*

  9. Cómo puedo borrar una rama?

    git branch -d | -D *branch_a_remover*

  10. Cómo puedo borrar un tag?

    git tag -d *tag_a_borrar*

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    "git status" o "git diff" para cambios unstaged

  12. Para que sirve el comando git remote -v?

    "git remote" permite manejar repositorios. Ya sea agregando, removiendo, modificando, etc. Y "-v" es verbose.

  13. Cómo le digo a git cual es mi email y correo?

    git config --global user.name "John Doe" git config --global user.email johndoe@example.com

  14. Cuales son los pasos para conectar git con github?

    cd /Users/MiUsuario/Desktop/github_folder $ git clone https://github.com/testuser/test.git $ cd test $ git status $ git add "prueba.txt" $ git commit -m "test" $ git remote -v $ git push origin master

  15. Cómo puedo crear un alias en git?

    git config --global alias.hola 'echo "hola"'
