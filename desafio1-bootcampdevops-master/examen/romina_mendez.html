Examen Git y Github

Alumn@: Romina Mendez

  1. Que es un sistema de control de versiones?

    Es un sistemas que permite a los equipos de desarrollo a poder gestionar los cambios en el codigo a lo largo del tiempo.

  2. Cual es la diferencia entre git y github?

    Git es un software que permite gestionar el codigo fuente, mientras que github es un servicio en la nube que permite cargar o desacargar recursos contenidos en repositorios

  3. Cómo creo una rama en git?

    Para crear una rama podemos utilizar `git branch ` o para crear una rama y saltar a ella se puede realizar con git checkout -b

  4. Cómo clono un repositorio?

    Para clonar un repositorio puedo utilizar podemos utilizar `git clone `

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    Para poder descargar los ultimos cambios realizados sobre el repositorio se debe colocar `git pull

  6. Cómo puedo gener un tag?

    Para poder generar un tag con la version se puede realizar `git tag v0.9.1 -m "este es un mensaje"`

  7. Cómo hago una mezcla de ramas?

    Para unificar dos ramas debo realizar git merge

  8. Cómo puedo moverme a un commit especifico?

    Para moverme en un commit especifico primero debo buscar el hash del commit con `git log` y posteriormente debo realizar `git checkout `

  9. Cómo puedo borrar una rama?

    Para borrar una rama debo en primer lugar cambiar de rama y posteriormente git branch -d

  10. Cómo puedo borrar un tag?

    Para borrar un tag se debe relizar ´git tag --delete `

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    Para ver los cambios que se estan por enviar a un repositorio se puede realizar un `git log --stat`

  12. Para que sirve el comando git remote -v?

    Permite ver cual son la Url que git tiene asociada al repo para escribir o leer.

  13. Cómo le digo a git cual es mi email y correo?

    Para configurar en git mi correo ectronico es `git config --global user.email

  14. Cuales son los pasos para conectar git con github?

    Para conectarlos, primero debo clonar un proyecto, posteriormente se añaden los cambios y se realiza un push para llevar los cambios al repositorio en github.

  15. Cómo puedo crear un alias en git?

    Para crear un alias podemos utilizar ´git config --global alias `
