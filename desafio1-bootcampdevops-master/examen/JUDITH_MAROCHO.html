Examen Git y Github

Alumn@: [Judith Marocho]

  1. Que es un sistema de control de versiones?

    [Es una herramienta de software que ayuda a los equipos de software a gestionar, de manera mas rápida, los cambios en el código fuente a lo largo del tiempo ]

  2. Cual es la diferencia entre git y github?

    [Que git sirve para controlar las versiones de los archivos mientras que GitHub es una plataforma para alojar los repositorios Git. ]

  3. Cómo creo una rama en git?

    [Me posiciono en la rama que equiero ramificar,por ejemplo master, y escribo el comando 'git branch nombre_nueva_rama' o 'git checkout -b nombre_nueva_rama']

  4. Cómo clono un repositorio?

    [Desde la consola abro la carpeta donde me quiero posicionar a trabajar y escribo el comando 'git clone origen_del_repositorio_a_clonar']

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [Con el comando 'git pull' si estoy seguro de bajar todos los cambios o si es un arama especifica 'git pull origin nombre_rama]

  6. Cómo puedo gener un tag?

    [Con el subcomando de Git 'git tag' por ejemplo 'git tag v0.x.x -m "mensaje_del_tag"']

  7. Cómo hago una mezcla de ramas?

    [Con el comando 'git merge' por ejemplo si quisiera subirlo a la rama master desde la rama en la que estamos actualmente, sería así 'git merge rama_con_cambios_a_mezclar']

  8. Cómo puedo moverme a un commit especifico?

    [Primero tenemos que ver nuestro log 'git log' para tomar nuestro numero de commit o hash y luego con ese numero ejecutar el comando 'git checkout numero_hash']

  9. Cómo puedo borrar una rama?

    [Con el comando 'git branch -d nombre_rama']

  10. Cómo puedo borrar un tag?

    [Para borrar un tag que no ha sido publicado, es decir que aun esta local, seria con 'git tag -d nombre_tag', de lo contrario, si ya se subio al servidor, seria con 'git push --delete origin nombre_tag']

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [Con el comando'git diff' ]

  12. Para que sirve el comando git remote -v?

    [Sirve para ver la url remota del origin del repositorio central, por ejemplo: la url de github.com donde esta alojado el repositorio tanto para jalar cambios (fetch), como para subirlos (push)]

  13. Cómo le digo a git cual es mi email y correo?

    [Entiendo que en email iba nombre. De ser asi iria con los siguientes comandos: 'git config --global user.name "Judith"' y 'git config --global user.email judith@hotmail.com']

  14. Cuales son los pasos para conectar git con github?

    [Con elcomando 'git remote add origin https://github.com/user/repo.git']

  15. Cómo puedo crear un alias en git?

    [Con el comando 'git config' , por ejemplo si quiera hacer un alias de status seria 'git config --global alias.st status']
