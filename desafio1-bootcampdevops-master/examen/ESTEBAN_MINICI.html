Examen Git y Github

Alumn@: [Esteban Minici]

  1. Que es un sistema de control de versiones?

    [Una metodologia para que, utilizando una aplicacion se logre tener un mejor control del versionado de algo, en este caso archivos para un desarrollo]

  2. Cual es la diferencia entre git y github?

    [Git es una aplicacion mientras que github es un repositorio en internet]

  3. Cómo creo una rama en git?

    [git checkout -b Nombredelarama]

  4. Cómo clono un repositorio?

    [git clone NombreOURLdelRepo]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull origin master]

  6. Cómo puedo gener un tag?

    [git tag -a nombredeltag ]

  7. Cómo hago una mezcla de ramas?

    [git merge nuevarama]

  8. Cómo puedo moverme a un commit especifico?

    [git checkout idelcommit]

  9. Cómo puedo borrar una rama?

    [git branch -d nombredelaramaaborrar]

  10. Cómo puedo borrar un tag?

    [git tag --delete nombredeltag]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]

  12. Para que sirve el comando git remote -v?

    [Para ver donde se encuentra el repositorio]

  13. Cómo le digo a git cual es mi email y correo?

    [Utilizando git config --global user.name y user.email]

  14. Cuales son los pasos para conectar git con github?

    [Genero mi repositorio en Github, luego voy a Git y clono ese repositorio con git clone]

  15. Cómo puedo crear un alias en git?

    [Utilizando git config --global alias.XX XXXXXXX donde XX reemplazara al comando XXXXXX cuando quieras utilizarlo, ejemplo git config --global alias.br branch luego cada vez q quiera utilizar el comando branch solo escribire git br]
