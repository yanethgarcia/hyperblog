Examen Git y Github

Alumn@: Isabella Serna

  1. Que es un sistema de control de versiones?

    [Herramienta que permite la gestión de cambios en un sistema de archivos para mejorar el flujo de trabajo entre los equipos de desarrollo]

  2. Cual es la diferencia entre git y github?

    [Git es un sitema de control de versiones que ayuda a realizar seguimiento del historial del código y Github es un servicio de alojamiento de repositorios]

  3. Cómo creo una rama en git?

    [Con el comando: git branch [nombre-rama] ]

  4. Cómo clono un repositorio?

    [Con el comando: git clone [url-repositorio] (ya sea por SSH o HTTPS)]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [Con el comando: git pull origin [nombre-rama]]

  6. Cómo puedo gener un tag?

    [Con el comando: git tag -a [nombre-tag] -m [mensaje]]

  7. Cómo hago una mezcla de ramas?

    [Con el comando: git merge [nombre-rama]]

  8. Cómo puedo moverme a un commit especifico?

    [Con el comando: git checkout [hash-commit]]

  9. Cómo puedo borrar una rama?

    [Con el comando: git branch -d [nombre-rama]]

  10. Cómo puedo borrar un tag?

    [Con el comando: git tag -d [nombre-tag]]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [Con el comando: git status o git diff]

  12. Para que sirve el comando git remote -v?

    [Muestra repositorios remotos]

  13. Cómo le digo a git cual es mi email y correo?

    [git config user.name "[username]" git config user.email [email]]

  14. Cuales son los pasos para conectar git con github?

    [git remote add origin [url]]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.co [alias]]
