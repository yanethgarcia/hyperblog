Examen Git y Github

Alumn@: [Patricia Cayo]

  1. Que es un sistema de control de versiones?

    [Es una herramienta que permite entre otras muchas cosas llevar un control del versionado cuando y quien hizo los cambios en los archivos, es muy practico, confiable y agiliza la tarea de entega de software]

  2. Cual es la diferencia entre git y github?

    [git es un sistemas de control de versiones mientras que github es una plataforma donde se alojan repositorios, es como una red social para desarrolladores..]

  3. Cómo cre una rama en git?

    [git branch nueva_rama]

  4. Cómo clono un repositorio?

    [git clone enlace_repositorio]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull]

  6. Cómo puedo gener un tag?

    [git tag -a v1.4 -m 'my version 1.4']

  7. Cómo hago una mezcla de ramas?

    [git merge origen rama]

  8. Cómo puedo moverme a un commit especifico?

    [git checkout nro_hash]

  9. Cómo puedo borrar una rama?

    [git branch -d rama]

  10. Cómo puedo borrar un tag?

    [localmente: git tag -d nombreDelTag remotamente:git push origin :nombreDelTag]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git diff [archivo] git diff --cached mostrara los cambios que se incluirán en el historial si se hace commit ]

  12. Para que sirve el comando git remote -v?

    [lista los remotos existentes]

  13. Cómo le digo a git cual es mi email y correo?

    [git config --global user.email email_usuario]

  14. Cuales son los pasos para conectar git con github?

    [1- agregamos el nuevo remoto: git remote add origin [URL de tu repositorio] 2- git add . agregamos los cambios 3 - git commit -m "Mi mensaje" 4 - git push -u origin master para que los cambios se reflejen en github]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.[alias] "[comando]" donde comando es el que queremos sustituir]
