Examen Git y Github

Alumn@: [Jose Figueroa]

  1. Que es un sistema de control de versiones?

    [Un sistema de control de versiones es una herramienta utilizada en el desarrollo de software para evitar el riesgo de conflictos que puedan surgir al trabajar en colaboración con otros equipos de desarrollo]

  2. Cual es la diferencia entre git y github?

    [Git sirve para controlar las versiones de los archivos mientras que GitHub es una plataforma para alojar tus repositorios Git]

  3. Cómo creo una rama en git?

    [git checkout -b nombre_rama]

  4. Cómo clono un repositorio?

    [git clone pueder por https y ssh]

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    [git pull origin nombre_rama]

  6. Cómo puedo gener un tag?

    [git tag -a nombre_tag]

  7. Cómo hago una mezcla de ramas?

    [git merge]

  8. Cómo puedo moverme a un commit especifico?

    [git revert]

  9. Cómo puedo borrar una rama?

    [git branch -d nombre_rama]

  10. Cómo puedo borrar un tag?

    [git tag -d nombre_tag]

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    [git status]

  12. Para que sirve el comando git remote -v?

    [Muestra los repositorios remoto]

  13. Cómo le digo a git cual es mi email y correo?

    [git config --global user.name; gitconfig --global user.email (Para configurar nombre y correo)]

  14. Cuales son los pasos para conectar git con github?

    [Se debe crear una llave en git localmente para conectarse vía ssh y luego se debe agregar en las configuraciones del usuario (SSH Keys) en la cuenta github]

  15. Cómo puedo crear un alias en git?

    [git config --global alias.alias comando]
