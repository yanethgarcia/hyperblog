Examen Git y Github

Alumn@: Benjamin Abel Sucasaire Huamani

  1. Que es un sistema de control de versiones?

    Es un conjunto de procesos que permite versionar texto plano, y nos da la posibilidad de trabajar en equipo.

  2. Cual es la diferencia entre git y github?

    Uno es la herramienta para realizar el versionado de texto, y el otro es uno de los repositorios remotos compatibles con git.

  3. Cómo creo una rama en git?

    creamos una rama con el comando "git branch [nombre_rama]"

  4. Cómo clono un repositorio?

    Comando: "git clone [link_repositorio]"

  5. Cómo puedo bajar los ultimos cambios de un repositorio?

    comando: "git pull [rama]"

  6. Cómo puedo gener un tag?

    comando: "git tag -a [tag] -m '[texto_tag]'"

  7. Cómo hago una mezcla de ramas?

    comando: "git merge [rama_a_mezclar]"

  8. Cómo puedo moverme a un commit especifico?

    comando: "git checkout [codigo_commit]"

  9. Cómo puedo borrar una rama?

    commit: "git branch -d [rama]"

  10. Cómo puedo borrar un tag?

    comando : "git tag -d [nombre_tag]"

  11. Cómo puedo ver los cambios que estan por enviarse al repositorio?

    comando: "git status"

  12. Para que sirve el comando git remote -v?

    sirve para ver el repositorio de origen

  13. Cómo le digo a git cual es mi email y correo?

    comandos para configurar name y email git config --global user.name [nombre] git config --global user.email [correo]

  14. Cuales son los pasos para conectar git con github?

    como primer paso es clonar el repositorio que se tenga en github, luego añadir los cambios a local y subirlos con push a nuestra repositorio remoto

  15. Cómo puedo crear un alias en git?

    Podemos crear alias con el siguiente comando "git config --global alias. " ejemplo: git config --global alias.co checkout git config --global alias.br branch git config --global alias.ci commit git config --global alias.st status
